Create Sample Barcode Request

Sample barcodes are used to label and track samples with unique Field Site IDs. If a regularly sampled location does not yet have a Field Site ID, please see the Create Field Site ID page for further instructions.

Sample barcodes can be requested by filling out the Request Barcode Labels form below. To prevent duplicate requests, please coordinate among regions and designate individuals responsible for requesting and receiving barcode labels. Sample barcodes require an existing Field Site ID.

Naming Convention for Sample Barcodes

The first seven characters of a sample barcode is the Field Site ID. If you do not yet have a Field Site ID, please see the Create Field Site ID page for further instructions.

pRR_SCC_YYm_CCCC

Where:

  • p = Grant or project code
  • RR = Watershed region code
  • S = System type code
  • CC = Two digit sequence, or number
  • YY = Two digit collection year
  • m = Sample material code
  • CCCC = Four digit sequence, or number
